 Frequently Asked Questions

                    
                    

                        

                            What is Error Correction Level (ECC)?
                            

                                
Error correction enables QR readers to scan the code even if parts of it are damaged, dirty, or covered.
                                   
    
                                       
  • Level L (7%): Best for simple, fast scanning without logos.
    • 
                                       
  • Level M (15%): The default balanced level.
    • 
                                       
  • Level Q (25%): Solid level for logos.
    • 
                                       
  • Level H (30%): Recommended when adding logos, overlays, or when printed in harsh conditions.
    • 
                                   

                                

                            

                        

                        
                        

                            Why is SVG download better than PNG?
                            

                                
SVG (Scalable Vector Graphics) is a vector format. This means it can be scaled infinitely to any print resolution without pixelating or blurring. Ideal for physical poster designs, banners, and flyers. PNG is a raster image format, which is better for web shares and app usage.

                            

                        


                        

                            How does contrast checking work?
                            

                                
QR code scanners rely on the contrast differences between the dark modules (foreground) and light modules (background). If you choose a foreground color that is too light or a background that is too dark, readers won't register the code. Our contrast badge monitors color values and alerts you if contrast falls below scannable levels.

                            

                        


                        

                            Is Chandra truly free?
                            

                                
Yes! Chandra is 100% free with unlimited generation, scanning, and high-quality vector downloads.
